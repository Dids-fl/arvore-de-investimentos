# Recomendador de Investimentos

Sistema interativo que, a partir de um questionário, recomenda produtos e ativos financeiros adequados ao perfil do usuário.

## Instalação

1. Clone o repositório.
   https://github.com/Dids-fl/arvore-de-investimentos.git

2. Crie um ambiente virtual:
   ```bash
   python -m venv .venv
   
   source .venv/bin/activate  # Linux/Mac
   .venv\Scripts\activate     # Windows
